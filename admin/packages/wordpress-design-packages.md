# پکیج‌های طراحی وب با وردپرس

## پکیج ساده - 15,000,000 تومان

### ویژگی‌ها:
- صفحات عادی
- طراحی عادی
- قالب وردپرس استاندارد
- بهینه‌سازی اولیه برای موتورهای جستجو
- طراحی ریسپانسیو (موبایل، تبلت، دسکتاپ)
- راه‌اندازی اولیه و آموزش پایه

### شامل:
- طراحی و راه‌اندازی تا 5 صفحه
- نصب و پیکربندی قالب
- اضافه کردن محتوای اولیه
- اتصال به شبکه‌های اجتماعی
- فرم تماس ساده

## پکیج متوسط - از 25,000,000 تومان به بالا

### ویژگی‌ها:
- طراحی قوی‌تر از پکیج ساده
- صفحات بیشتر
- گرافیک حرفه‌ای‌تر
- قالب وردپرس اختصاصی یا پریمیوم
- بهینه‌سازی پیشرفته برای SEO
- طراحی ریسپانسیو کامل
- پشتیبانی و آموزش جامع

### شامل:
- طراحی و راه‌اندازی تا 10 صفحه
- نصب و سفارشی‌سازی قالب پریمیوم
- طراحی لوگو و برندینگ
- گالری تصاویر
- وبلاگ با دسته‌بندی
- فرم‌های پیشرفته
- یکپارچه‌سازی با شبکه‌های اجتماعی
- افزونه‌های ضروری
- بهینه‌سازی سرعت
- پشتیبانی 3 ماهه

- پشتیبانی کامل و آموزش جامع
- سئو تکنیکال و محتوا

### شامل:

- پشتیبانی 6 ماهه

- بهینه‌سازی برای موتورهای جستجو (SEO)

### مدت زمان تحویل:
- پکیج ساده: 7-10 روز کاری

- پکیج متوسط: 3 ماه پشتیبانی رایگان
